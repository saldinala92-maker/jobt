## Features

- **Matched for You:** Personalized job recommendations based on your skills.

## UI Design
- **Dark Mode Architecture:** Elegant dark theme with vibrant accents (Lime, Sky, Lavender).
- **Mobile First:** Designed for a 393x852 viewport (iPhone 14 Pro).
- **Interactivity:** Smooth animations, hover effects, and custom-styled components.
- **Typography:** Built with 'Syne' and 'Epilogue' fonts for a modern look.
